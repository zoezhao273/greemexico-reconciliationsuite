How to use

  • 1
    Export the monthly NCC Bank Journal (.xlsx) from your accounting system. Export the ICBC Bank Statement (.csv, UTF-16) and optionally the Santander Bank Statement (.csv, UTF-8) from each bank portal.
  • 2
    Upload both the NCC Bank Journal and a Bank Statement — both are required. For the Bank Statement, upload either the ICBC file or the Santander file, not both at the same time. Uploading a second bank file will automatically clear the first.
  • 3
    Click Run Reconciliation. The tool auto-detects formats, skips subtotal rows, sorts by amount, and compares line by line.
  • 4
    Preview the discrepancies on screen, then click Download Excel to save the reconciliation report with two worksheets.
  • 5
    Yellow highlight = entry missing on one side. Green = matched. The Note column explains each discrepancy.
